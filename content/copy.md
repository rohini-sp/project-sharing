# Brand & copy notes

Messaging, tone, and key points pulled from the live sites
(https://xo.builders and https://docs.xo.builders). Agents: capture what you learn
here so the others can reuse it.

## Tagline

- Primary: **"SuperCharge your AI"**
- Hero value line: **"Launch your AI agent in 1 click"** (site uses "Launch Clawdbot in 1 Click")
- Supporting promise: **"Zero DevOps. Zero Downtime. Just deploy."**
- Docs one-liner: *"Secure, isolated remote environments for your AI agents. No local
  setup, no infra to manage — pick a template and your agent is live."*

## Key value props

- **Deploy in under 60 seconds** — no DevOps expertise required.
- **One-click, multi-channel** — Telegram, WhatsApp, Discord, Slack, iMessage, Web Chat.
- **Secure & isolated** — sandboxed containers, encrypted API keys, private workspaces,
  isolated memory/credentials per session. "Secure by Default." SOC2 Ready.
- **Multi-agent** — hot-swap between models; agent templates (OpenClaw, Claude Code,
  Hermes, XO Cowork).
- **Fully managed** — auto-scaling, patching, monitoring included. "Zero Downtime."
- **Open foundation** — built on open-source OpenClaw (200k+ GitHub stars).
- **Three-step deploy** — create a project → pick a template → launch.

## Calls to action

- Primary CTA: **"Try Now for Free"**
- Variant: **"Try 30 days for Free"**

## Tone / voice

Confident, developer-friendly, and direct. Short punchy lines ("Just deploy."),
technical credibility delivered in accessible language, benefit-first framing that
contrasts XO's simplicity against manual DevOps pain. Empowering, modern, no fluff.

## Brand design tokens (from xo.builders)

- **Font:** Inter (sans-serif).
- **Accent green:** `#83d63a` (lime) / `#22c55e` (green) — primary brand color.
- **Cyan accent:** `#06b6d4` / `#6DD4F5` (secondary).
- **Ink / dark:** `#1F1F1E`, `#0a0a0a`, `#111111`.
- **Light bg / surfaces:** `#F8F8F8`, `#ffffff`, `#e6e6e6` borders.
- **Muted text:** `#616161` / `#757575`.
- Visual style: clean, modern, dashboard-centric, minimalist with monospace code accents.
